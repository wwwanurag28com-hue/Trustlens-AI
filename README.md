# 🕵️‍♂️ Trustlens-Ai (RAWCIT | D.F.D)
**Digital Forensics Department - Media & Text Verification System**

### 📌 About The Project
Trustlens-Ai is an advanced, AI-powered web intelligence portal designed to combat digital misinformation. It acts as a comprehensive digital forensics tool capable of detecting deepfake videos, AI-generated/manipulated images, and fake news text using state-of-the-art Machine Learning models and Computer Vision techniques. 

### 🚀 Key Features
* **Image Forensics:** Detects synthetic pixels and manipulated imagery using Cloud AI and Offline Laplacian Pixel Noise Analysis.
* **Video Forensics (Deepfakes):** Frame-by-frame temporal analysis using OpenCV to identify deepfake anomalies.
* **Text & News Forensics:** Integrates Tesseract OCR for screenshot reading and BERT NLP models for semantic fake news classification.
* **Zero-Guesswork Architecture:** Implements a strict patient-retry mechanism ensuring 100% reliable forensic verdicts.

### 💻 Technology Stack
* **Frontend:** HTML5, CSS3, JavaScript (Intelligence Agency UI/UX)
* **Backend:** Python 3, Flask Web Framework
* **Computer Vision:** OpenCV (cv2), Pillow (PIL)
* **Machine Learning:** Hugging Face Inference APIs (Vision Transformers, BERT)
* **OCR:** Google Tesseract
